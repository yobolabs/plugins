# Exemplars — Your Best War-Games

Accepted war-games that reality later PROVED accurate (via the after-action section or a simulation comparison). They are the standard the reviewer holds new drafts to, and the examples new team members learn from.

Empty on day one — that's expected. Populate via:
1. The `simulate` skill (blind war-games of past campaigns that held up against real results)
2. Live campaigns whose war-game predictions proved out in after-action

Rules:
- Only war-games verified against reality — being well-written isn't enough.
- Each entry: campaign name, date, one line on what it predicted correctly, one line on what it still missed (the honest asterisk).
- Never edit an exemplar to look smarter than it was — the team learns from true baselines.
