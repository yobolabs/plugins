# The Better Rubric — Marketing Edition

"Better" never means "more complicated." A campaign, flow, segment, or creative improves only along these axes:

| Axis | Question | Signals |
|------|----------|---------|
| **Clearer** | Does the customer instantly get the offer and what to do? | One message, one CTA; no explaining the explanation |
| **More on-brand** | Would the merchant proudly read this aloud? | Voice rules kept; no claim the brand can't back |
| **Better targeted** | Right people, right moment — not just more people? | Segment matches the offer's natural audience; fatigue respected |
| **More measurable** | Will we KNOW if it worked? | Trackable links/UTMs, redemption codes, a holdout slice where it matters |
| **Safer** | Deliverability, platform rules, promises we can keep | No spam-bait wording, tier limits respected, stock confirmed |
| **More reusable** | Does this work leave a template behind? | Saved segment, caption formula, drip skeleton usable next time |

## Rules

- Score only the axes a change moves; don't ritually grade all six.
- Every plus names its cost if one exists ("better targeted, less reach — the send is 40% smaller: worth it because…").
- Ties break toward the brief's primary axis.
- Reject on sight: an "improvement" that only adds steps, segments, or cleverness without moving any axis — complexity is a cost, not an achievement.
