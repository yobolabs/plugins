# Exemplar — merchant activation, SMB SaaS (Fable-direct)

- **mission class:** design-surface · activation (signup → first real value) · SMB marketing/CRM SaaS, low-technical-patience segment
- **lens set:** friction-cro + hooks-habit + incentives-game-theory (social-viral, pricing-economics declared out: activation of an existing signup is neither a k-factor nor a paywall problem)
- **outcome classes:** 9 candidates → 2 kills, 2 dark-pattern exclusions (S1–S2), 7 advanced, top-3 experiment cards; re-runs 0

## Distilled judge standards (what "good" looked like)

1. **Cliff decomposition before mechanics.** The setup-heavy funnel was decomposed into named cliffs (empty workspace / channel connection / blank composer) ordered by expected severity for THIS segment — every candidate then attacked a named cliff. Mechanics without a cliff address are decoration.
2. **The aha re-located ahead of setup.** The top card delivered felt value (a real message on the merchant's own phone) BEFORE the setup the value normally sits behind — recognizing that demo-value can precede data-value, with a guardrail (setup completion) against the demo becoming the terminal state. That guardrail choice is the standard: the mechanic's own best failure mode became its guardrail.
3. **Segment psychology gated broadcast mechanics.** Leaderboard kill: the segment is not proud of setup speed — broadcast preconditions checked against the recon psychology, not assumed from the mechanic's success elsewhere.
4. **Nudges contextual, never broadcast.** Stall-triggered targeted help advanced; drip sequences didn't appear; shame-framed nudges ("competitors already sent 3") explicitly flagged S1 rather than silently omitted — the line is drawn in the artifact, visibly.
5. **Activation definition itself flagged as work.** The brief refused to hard-code an activation threshold, citing the behavioral-derivation method (retained-cohort behavior, causation-checked) as a dependency on the instrumentation card — deriving the target is part of the mission, not an assumption.
6. **Banned-axis discipline inside guardrails.** A tempting guardrail (time-in-product) was rejected as a banned axis and replaced with resentment/gaming signals (meter dismissal, step-order integrity). <!-- validator:allow — names the ban -->
